# IDEA's toolbox

Utility functions to support iter IDEA's projects.

## Installation

`npm install idea-toolbox`

## Usage

Javascript

```
const IdeaX = require('idea-toolbox');
```

Typescript

```
import IdeaX = require('idea-toolbox');
```
